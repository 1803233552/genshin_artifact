## 简介
《原神》圣遗物自动搭配
## 特点
+ 支持所有角色以及所有武器，精确到个位数的等级
+ 不同角色使用不同养成方案
+ 纯前端，本地也可以使用
+ ...
## 使用策略
- 可以查看你的圣遗物能够最高堆到多少攻击、暴击率、暴击伤害等
- 我的圣遗物能到达多少期望伤害？
- 诺艾尔的防御和攻击怎么平衡？
- ...
## 本地运行
该网站没有后端，可以本地运行
```
npm run serve
```
